# 🎮 经典游戏合集 Classic Games Collection

欢迎来到经典游戏合集！这是一个精心制作的网页游戏平台，包含了多个经典的街机和益智游戏。所有游戏都经过优化，支持现代浏览器和各种设备。

## ✨ 特色功能

- 🎯 **多游戏支持** - 集成多个经典游戏
- 📱 **响应式设计** - 支持桌面、平板、手机
- ⌨️ **多种控制方式** - 方向键 + WASD 支持
- 🚀 **高性能** - 优化高刷新率屏幕
- 🎨 **现代化UI** - 美观的界面和动画效果
- 📊 **实时监控** - FPS显示和性能统计

## 🎮 包含游戏

### 2048 数字游戏
经典的数字合并益智游戏，通过滑动合并相同数字，最终达到2048！

- **游戏类型:** 策略益智
- **控制方式:** 方向键/WASD
- **特色功能:** 本地高分记录、触摸支持、动画效果

### 吃豆人游戏
经典的街机游戏！控制吃豆人在迷宫中收集豆子，躲避幽灵的追捕。

- **游戏类型:** 街机动作
- **控制方式:** 方向键/WASD
- **特色功能:** 幽灵AI、能量豆、高分系统、FPS显示

### 康威生命游戏
经典的细胞自动机游戏！观察细胞按照简单规则演化，创造出复杂的生命模式。

- **游戏类型:** 数学模拟
- **控制方式:** 鼠标点击/触摸绘制
- **特色功能:** 预设图案、实时统计、可调节速度

### 贪吃蛇游戏
经典的贪吃蛇游戏！控制小蛇吃食物成长，避免撞墙和咬到自己，挑战更高分数和更长的蛇身。

- **游戏类型:** 街机经典
- **控制方式:** 方向键/WASD
- **特色功能:** 多难度选择、等级系统、本地高分记录、移动端支持

## 🚀 快速开始

### 在线游玩
访问GitHub Pages网站直接在线游玩：
**https://bonana521.github.io/2048-game/games_home.html**

### 本地运行

1. **克隆仓库**
   ```bash
   git clone https://github.com/bonana521/2048-game.git
   cd 2048-game
   ```

2. **打开游戏**
   ```bash
   # 方法1: 直接在浏览器中打开 index.html 或 games_home.html
   # 方法2: 使用本地服务器 (推荐)
   python -m http.server 8000
   # 然后访问 http://localhost:8000/
   ```

3. **选择游戏**
   - 打开 `index.html` 会自动跳转到游戏主页
   - 选择你想要游玩的游戏
   - 使用键盘或触摸控制游戏

## 🎯 游戏控制

### 2048游戏
- **移动:** ↑↓←→ 方向键 或 W/A/S/D 键
- **新游戏:** 点击"新游戏"按钮
- **返回主页:** 点击"返回主页"按钮

### 吃豆人游戏
- **移动:** ↑↓←→ 方向键 或 W/A/S/D 键
- **开始/暂停:** 空格键
- **返回主页:** 点击"返回主页"按钮

### 康威生命游戏
- **绘制细胞:** 鼠标点击或触摸屏幕
- **开始/暂停:** 点击"开始"/"停止"按钮
- **单步演化:** 点击"单步"按钮
- **清空画布:** 点击"清空"按钮
- **随机生成:** 点击"随机"按钮
- **调节速度:** 修改速度输入框（毫秒）
- **加载图案:** 点击预设图案按钮
- **返回主页:** 点击"返回主页"按钮

### 贪吃蛇游戏
- **移动:** ↑↓←→ 方向键 或 W/A/S/D 键
- **开始/暂停:** 空格键
- **新游戏:** 点击"重置游戏"按钮
- **难度选择:** 选择难度等级（简单/普通/困难/极限）
- **返回主页:** 点击"返回主页"按钮

### 主页快捷键
- **数字键 1:** 快速跳转到2048游戏
- **数字键 2:** 快速跳转到吃豆人游戏
- **数字键 3:** 快速跳转到生命游戏
- **数字键 4:** 快速跳转到贪吃蛇游戏

## 🛠️ 技术栈

### 前端技术
- **HTML5** - 语义化标记和Canvas API
- **CSS3** - Flexbox、Grid、动画、渐变
- **JavaScript (ES6+)** - 原生JS，无框架依赖
- **响应式设计** - 移动端优化

### 游戏引擎
- **Canvas 2D** - 高性能图形渲染
- **RequestAnimationFrame** - 流畅动画循环
- **自定义游戏引擎** - 轻量级游戏逻辑管理

### 性能优化
- **基于时间的移动控制** - 独立于刷新率
- **事件节流** - 优化用户输入处理
- **内存管理** - 高效的对象复用
